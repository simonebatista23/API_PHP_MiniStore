# API_PHP_MiniStore

**API_PHP_MiniStore** é uma API desenvolvida em PHP para uma mini loja online, oferecendo funcionalidades essenciais para gerenciamento de produtos, pedidos e promoções. Ideal para integrar com frontends em HTML/JS, React ou qualquer outra tecnologia.

## Funcionalidades

- Gerenciamento de **produtos** (listar, adicionar, editar, remover)
- Controle de **carrinho de compras**
- Aplicação de **cupons de desconto**
- Criação e gerenciamento de **pedidos**
- Retorno de dados em **JSON**, pronto para integração com front-end
- Estrutura simples e fácil de estender

## Tecnologias

- PHP 8+
- MySQL
- Composer (para dependências)
- Servidor web (XAMPP)
